# Usage

### Starting the server

```
gradle server --args "port"
```

### Starting the client

```
gradle client --args "username serverip serverport"
```

# Example

### Starting the server

```
gradle server --args "5050"
```

### Starting the client

```
gradle client --args "moritz localhost 5050"
```
